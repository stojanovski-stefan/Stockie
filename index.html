Web Applications Programming

CSE383 B

The Course:

CSE 383, Web Applications Programming, is a semester long course offered at Miami University. As part of the computer science core, CSE 383 teaches conventional web development strategies and methodologies. The course content consists of, in order: Linux, HTML, CSS, Forms, Bootstrap, Javascript, Jquery, API, Ajax, Cloud servers, PHP, and security. Every week touches on a new topic, and includes new labs and homework assignments.

The Project:

Our final project, the stock application, is a culmination of every methodology we've learned throughout the course. It is not only meant to demonstrate our understanding of fundamental web development concepts, but also our ability to apply it. This application handles user logins, sign-ups, and logouts, and presents a comfortable and convenient interface for managing exchanges and involved stocks. Users are able to freely search through various exchanges and stocks, and are provided with charts, rates, trends, data, and news relating to each stock. The site allows users to add stocks to their favorites list, so as to provide convenient access to them in the future. The user can also view a log of which stocks they've added or removed from their favorites, which updates by the day. Lastly, the application provides a search bar where users can search stocks by ticker if they have specific stocks that they want to find. Thank you for viewing our project!
